DEDIER: Mathematical Approach

\[ \text{Student: } f_\theta = h \circ g_d,\quad \mathbf{p}^{(\text{aux})}(x) = h_{\text{aux}}(g_d(x)),\quad \hat{y}^{(\text{aux})}(x) = \arg\max \mathbf{p}^{(\text{aux})}(x) \]
\[ \text{Confidence margin: } \operatorname{cm}(\mathbf{p}) = \max_k p_k - \max_{k'\neq \arg\max p_k} p_{k'} \]
\[ \text{Instance weight: } w(x)= \begin{cases} 1, & \hat{y}^{(\text{aux})}(x)=y \\[4pt] \exp\!\big(\beta \cdot \operatorname{cm}(\mathbf{p}^{(\text{aux})}(x))^{\alpha}\big), & \text{otherwise} \end{cases} \]
\[ \mathcal{L}_{\text{DEDIER}} = \sum_{(x,y,w)} \Big[ (1-\lambda)\,\ell_{\text{ce}}(y^{(S)},y) \;+\; \lambda\, w\, \ell_{\text{kd}}(x) \Big] \] \[ \ell_{\text{kd}}(x)=\tau^2\,\text{KL}\big(\sigma(y^{(S)}/\tau)\,\|\,\sigma(y^{(T)}/\tau)\big) \]
Algorithm (short):
Initialize \(w_i \leftarrow 1\).
For each epoch \(e\):
  1. Update \(\theta\) with \(\nabla\mathcal{L}_{\text{DEDIER}}\)
  2. If \(e \bmod L = 0\):
     - Retrain \(h_{\text{aux}}\) on \(\{(g_d(x_i), y_i)\}\)
     - Recompute \(w_i\) via \(w(x)\) above
Output: trained student \(f_\theta\).

Variables & Symbols

VariableDescription
\(x\)Input instance
\(y\)True label
\(f_{\theta}\)Student model parameterized by \(\theta\)
\(\theta\)Parameters of the student model
\(g_d\)Early layers up to depth \(d\) (feature extractor)
\(d\)Depth index for early readout (hyperparameter)
\(h\)Deeper layers after \(g_d\)
\(h_{\text{aux}}\)Auxiliary classifier (early readout head)
\(\mathbf{p}^{(\text{aux})}(x)\)Softmax output of auxiliary classifier
\(\hat{y}^{(\text{aux})}(x)\)Predicted class from early readout
\(\operatorname{cm}(\mathbf{p})\)Confidence margin: \(\max p_k - \max_{k'\neq \arg\max p_k} p_{k'}\)
\(w(x)\)Instance‑specific weight for distillation loss
\(\alpha\)Hyperparameter for margin sensitivity (\(\alpha>0\))
\(\beta\)Hyperparameter for weight scaling (\(\beta>0\))
\(\mathcal{D}_w\)Augmented dataset \(\{(x_i, y_i, w_i)\}\)
\(\ell_{\text{ce}}\)Cross‑entropy loss with true labels
\(\ell_{\text{kd}}(x)\)Distillation (teacher‑matching) loss
\(y^{(S)}(x)\)Student logits \(f_{\theta}(x)\)
\(y^{(T)}(x)\)Teacher logits \(\mathcal{T}(x)\)
\(\sigma(\cdot)\)Softmax function
\(\tau\)Temperature parameter
\(\lambda\)Balance between cross‑entropy and distillation (\(\lambda \in [0,1]\))
\(L\)Retraining interval (epochs)
\(R\)Number of epochs to retrain \(h_{\text{aux}}\)
\(E_{\max}\)Total training epochs
\(\eta\)Learning rate

\(\alpha,\beta,\lambda,d,L,R,\tau,\eta\) are hyperparameters tuned via validation worst‑group accuracy.
